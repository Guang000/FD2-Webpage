# FD² Project Page

Project website for **FD²: A Dedicated Framework for Fine-Grained Dataset Distillation**, accepted at **ECCV 2026**.

[Paper](https://arxiv.org/abs/2603.25144) · [Live site](https://guang000.github.io/FD2-Webpage/)

## Publish with GitHub Pages

In the repository settings, open **Pages**, choose **Deploy from a branch**, then select the `main` branch and `/ (root)`.

## Local preview

Open `index.html` directly, or serve this directory with any static web server.
